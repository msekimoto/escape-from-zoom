# Escape From Zoom

Um jogo cooperativo de fuga em um zoológico, feito para Roblox.

## Conceito
Os jogadores controlam animais tentando escapar de um zoológico enquanto completam objetivos, evitam guardas e usam habilidades únicas.

## Personagens
- Elly
- Leon
- Porinha
- Momo
- Snapper
- Grumblet

## Controles
Q = habilidade 1
E = habilidade 2
